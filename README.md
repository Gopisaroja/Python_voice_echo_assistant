# 🎙️ Python Speech Recognition

A Python-based speech recognition application that converts spoken words into text and responds using text-to-speech. The application continuously listens for voice input, repeats the recognized text, handles common recognition errors gracefully, and exits with a voice command.

## ✨ Features

* 🎤 Real-time speech recognition
* 🔊 Text-to-speech voice responses
* 🔁 Continuous voice listening
* ⚠️ Exception handling for recognition errors
* 🚪 Voice command to exit the application

## 🛠️ Tech Stack

* Python
* SpeechRecognition
* pyttsx3
* PyAudio

* Working with speech recognition in Python

* Handling microphone input
